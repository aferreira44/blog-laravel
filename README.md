# A Blog made with Laravel

## Getting Started

1. Install npm dependencies

    `npm install`

1. Copy `.env.example` to `.env` and set up MySQL connection

1. Run Migrations Database

    `php artisan migrate`

1. Run application

    `php artisan serve`

1. Access [http://127.0.0.1:8000](http://127.0.0.1:8000) on your browser
